Linux系统ifconfig查不到IP

2020/10/20

ubuntu使用ifconfig突然查不到IP了，只有lo没有ens33

用ifconfig -a可以看到ens33但是没有IP地址

使用ifconfig ens33 up启动ens33网卡后，再ifconfig可以看到ens33，有IPv6地址但是没有IPv4地址

然后使用一下两条命令之后终于可以看到IPv4地址了

sudo dhclient ens33

sudo ifconfig ens33

同时使用ifconfig也可以了

修改netplan配置文件，设置对应的网卡（ens33)使用DHCP获取IP或配置静态IP

sudo vim /etc/netplan/01-network-manager-all.yaml
sudo netplan apply

network:
  ethernets:
    ens33:     #配置的网卡的名称
      addresses: [192.168.2.134/24]    #配置的静态ip地址和掩码
      dhcp4: no    #关闭DHCP，如果需要打开DHCP则写yes
      optional: true
      gateway4: 192.168.2.2    #网关地址
      nameservers:
         addresses: [192.168.2.2,114.114.114.114]    #DNS服务器地址，多个DNS服务器地址需要用英文逗号分隔开
  version: 2
  renderer: networkd    #指定后端采用systemd-networkd或者NetworkManager，可不填写则默认使用systemd-workd

注意事项：
1、ip地址和DNS服务器地址需要用[]括起来，但是网关地址不需要
2、注意每个冒号后边都要先加一个空格
3、注意每一层前边的缩进，至少比上一层多两个空格
